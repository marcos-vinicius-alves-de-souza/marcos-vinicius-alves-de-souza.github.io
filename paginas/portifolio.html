O que eu já fiz?

Atualmente possuo 2 projetos principais que já publiquei e estão para uso de todos, vou exemplificar abaixo ambos!

Crabfetcher

Este projeto é uma alternativa feita em linguagem de programação Rust ao comando de terminal geralmente utilizado em distribuições Linux "neofetch".

O maior desafio enfrentado aqui foi obter o conhecimento necessário na linguagem para conseguir fazer a requisição das informações ao sistema operacional. Outro desafio também foi exibir a arte em ASCII, porém, sem muitos problemas eu consegui exibir ela.

Link para o github: https://github.com/marcos-vinicius-alves-de-souza/crabfetcher

Awesome Tarot

Este projeto já é de um teor um pouco mais pessoal, contudo, achei válido incluir ele aqui pois é um projeto público, aqui o objetivo é ser um site online de tarot que permita com que as pessoas consigam definir quais naipes estão inclusos, quantos embaralhamentos estão disponíveis e etc. Atualmente não se encontra completo porque eu ainda preciso incluir outras features, mas, é um site que se encontra bem desenvolvido no meu ponto de vista e conheço pessoas que o usam para fins pessoais e não comerciais.

Link para o github: https://github.com/awesome-tarot/rider-waite

Os dois projetos possuem em comum um desafio, em ambos eu não havia a habilidade necessária pronta para termina-los de uma vez, porém, através de esforço e dedicação eu sabia que conseguiria completa-los e através desse foco em entregar esses projetos, por mais que pessoais, eu entendi que ser um desenvolvedor muitas das vezes é sobre isso, ter alguma coisa e encontrar o que falta para completar com sucesso a tarefa.
